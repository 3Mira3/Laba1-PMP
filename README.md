# Currency Savings Calculator — Android App

Android-додаток для розрахунку заощаджень у іноземній валюті (євро або долар) протягом року.  
Додаток використовує курси валют на початок та кінець року, виконує всі необхідні обчислення та показує фінальний результат.

## Функціональність
- Головний екран з кнопками **"Почати розрахунок"** та **"Вийти"**
- Введення:
  - місячного доходу (M)
  - відсотка (p), який конвертується у валюту (0 < p < 1)
- Вибір валюти: **EUR** або **USD**
- Розрахунки (реалізовані у `CurrencyCalculator.kt`):
  - річний гіпотетичний дохід (Sy)
  - сума для обміну (Sc)
  - кількість придбаної валюти (W)
  - гривнева вартість валюти на кінець року (SH)
  - гривневий залишок (SL)
  - підсумкова сума H = SH + SL
  - результат заощадження R = H − Sy
- Екран результатів з кнопкою **"Ок"** для повернення на головний екран
- Підтримка портретної та альбомної орієнтації (`layout` + `layout-land`)
- Збереження стану при повороті екрану (`onSaveInstanceState`)
- Навігація між екранами через `Activity` та `Intent`

## Як запустити проєкт локально

### 1. Клонувати репозиторій
```bash
git clone https://github.com/3Mira3/Laba1-PMP.git
cd Laba1-PMP
```
### 2. Відкрити в Android Studio

- Вибрати File → Open

- Вибрати папку проєкту

### 3. Встановити необхідні SDK та залежності

Android Studio автоматично запропонує встановити потрібні компоненти (compileSdk, platform tools тощо).

### 4. Запустити

- Створити або вибрати пристрій (емулятор / фізичний смартфон)

- Натиснути Run ▶

### Вимоги

- Android Studio (Koala / Ladybug або новіша)

- JDK 17

- compileSdk = 36

- Мінімальна версія Android: API 24+
