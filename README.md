Currency Savings Calculator — Android App

Цей проєкт — Android-додаток для розрахунку заощаджень у іноземній валюті (євро або долар) протягом року.
Додаток використовує дані про курси валют на початок та кінець року, обчислює річний дохід, суму обміну, кількість придбаної валюти та фінальний результат економії.

Функціональність

Головний екран з кнопками “Почати розрахунок” та “Вийти”.

Введення місячного доходу та відсотка, який конвертується в валюту.

Вибір валюти (EUR або USD).

Розрахунок:

гіпотетичного річного доходу,

витрат на конвертацію,

кількості придбаної валюти,

її вартості на кінець року,

кінцевого балансу та результату заощадження.

Підтримка портретної та альбомної орієнтації через окремі layout-файли.

Збереження стану при повороті екрану (onSaveInstanceState).

Навігація між екранами через Activity (Intent + extras).

Структура проєкту

Проєкт складається з 4 основних Activity:

MainActivity

Calculation1Activity

Calculation2Activity

ResultsActivity

Та класу логіки розрахунків:

CurrencyCalculator.kt

Інтерфейс побудовано на ConstraintLayout (без Compose).
Усі екранні макети представлені у res/layout та res/layout-land.

Як запустити проєкт локально
1. Клонувати репозиторій
git clone https://github.com/3Mira3/Laba1-PMP.git

2. Відкрити проєкт

Запустіть Android Studio

Оберіть File → Open…

Вкажіть папку з проєктом Laba1-PMP

3. Встановити необхідні SDK

Android Studio запропонує встановити версії:

compileSdk = 36

targetSdk = 36

Android Gradle Plugin (AGP) 8.x

Gradle 8.x

Погодьтеся на встановлення, якщо буде потрібно.

4. Запустити додаток

Підключіть реальний Android-пристрій або створіть емулятор (AVD).

Натисніть Run ▶ у Android Studio.

Виберіть пристрій для запуску.

Після збірки додаток автоматично встановиться й відкриється.

Вимоги

Android Studio Ladybug / Koala або новіша

JDK 17 (встановлюється разом з Android Studio)

Мінімальна версія Android: API 24+
